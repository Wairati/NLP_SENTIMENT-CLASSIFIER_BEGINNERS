# NLP_SENTIMENT-CLASSIFIER_BEGINNERS
Building a simple NLP classifier.
Dataset: Any small text dataset (Twitter tweets)
Method:
1. Load dataset
2. Clean text (lowercase, remove punctuation, stopwords)
3. Convert text to features (Bag-of-Words or TF-IDF)
4. Train a classifier (Naive Bayes, Logistic Regression, or SVM)
5. Evaluate accuracy
6. Test on new sentences
